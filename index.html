⚙️ Guida Firebase

1. Vai su console.firebase.google.com

2. Crea un nuovo progetto (gratuito)

3. Vai su Build → Realtime Database → Crea database → scegli "Modalità test"

4. Vai su Impostazioni progetto → Le tue app → Aggiungi app Web

5. Copia l'oggetto firebaseConfig e incollalo in index.html sostituendo i valori INSERISCI_*

6. Vai su Realtime Database → Regole e imposta:

{
  "rules": {
    ".read": true,
    ".write": true
  }
}

⚠️ Per uso personale è ok. Per maggiore sicurezza, usa Firebase Auth.
